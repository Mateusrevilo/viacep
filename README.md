# ViaCEP - Cadastro de Usuario

Projeto simples para preencher automaticamente endereco a partir do CEP,
usando a API publica do ViaCEP, e salvar os dados no navegador.

## Funcionalidades

- Busca de endereco por CEP
- Preenchimento automatico dos campos
- Persistencia dos dados no `localStorage`

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Digite um CEP com 8 digitos e saia do campo.
3. Os campos de endereco serao preenchidos automaticamente.
4. Clique em "Enviar" para salvar os dados no navegador.

## Estrutura

- `index.html`: pagina principal
- `style.css`: estilos do formulario
- `script.js`: logica de busca e salvamento

## Tecnologias

- HTML
- CSS
- JavaScript
- API ViaCEP
